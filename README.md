# X-Ray Project

A full-stack web application initialized with:
- **Backend**: Node.js, Express, TypeScript
- **Frontend**: React, Redux, TypeScript, TailwindCSS

## Prerequisites

- Node.js (v16 or higher recommended)
- npm

## Getting Started

### 1. Backend Setup

The backend runs on port `3000`.

```bash
cd backend
npm install
npm run dev
```

### 2. Frontend Setup

The frontend runs on port `5173` and acts as a proxy to the backend.

```bash
cd frontend
npm install
npm run dev
```

## Structure

- `backend/`: Express server code
- `frontend/`: React application code
